# Contribuindo

Obrigado pelo interesse em contribuir com este projeto da Guebly.

## Fluxo de trabalho

1. Crie uma branch a partir de `main`: `git checkout -b feat/minha-feature`
2. Faça suas alterações com commits descritivos em português
3. Certifique-se de que o projeto builda sem erros
4. Abra um Pull Request descrevendo o que foi feito e por quê

## Padrões

- Commits em português, concisos e descritivos
- Sem `console.log` ou código comentado no merge final
- PRs pequenos e focados — um problema por vez

## Dúvidas

Fale com o time pelo canal interno ou abra uma issue.
